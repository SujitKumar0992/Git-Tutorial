## 3. Git status
```

```
Example
![](/1.Git Status.PNG)

## 4. Git Add
```
git add .
```
Adds all the current changes to the staging area.

```
git add <filename>
```
Adds the specic file to the staging area.

![](./2.Git add.PNG)

## 6. Push

```

```
Example
